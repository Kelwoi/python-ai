Вимоги до стилю коду:

- Використовуй async/await там, де це доречно.
- Увімкни та враховуй nullable reference types.
- Додай XML comments для публічних класів, інтерфейсів і методів.
- Дотримуйся clean architecture.
- Використовуй dependency injection для сервісів.

Правила генерації:

- Не змішуй бізнес-логіку з логікою введення/виведення.
- Додай обробку помилок.
- Якщо використовується ASP.NET Core, покажи реєстрацію сервісів у DI-контейнері.
- Якщо потрібні NuGet-пакети, вкажи команди для встановлення.
